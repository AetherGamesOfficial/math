# AETHER Study Hub

AETHER Study Hub is an education-themed web portal with study resources,
enrichment activities, and a static research browser powered by Sandstone.

## Pages

- `studyhub.html` - main study dashboard
- `research.html` - research browser
- `resources.html` - learning resources
- `enrichment.html` - enrichment activities

## Running Locally

Install dependencies:

```sh
pnpm install
```

Start the server:

```sh
pnpm start
```

Open:

```text
http://localhost:8080/studyhub.html
```

## Render

Use these commands:

```sh
pnpm install
```

```sh
pnpm start
```

## Static CDN Version

This folder is built to work as a static jsDelivr/GitHub Pages style copy.
Open `studyhub.html` as the entry page.
